# **App Name**: Fresh

## Core Features:

- Tree Owner Listing & Management: Tree owners can register, capture location, and list their fruit trees with essential details (type, quantity, photos) and preferred harvest dates for contractors.
- Contractor Bidding & Dashboard: Verified contractors can view available tree listings within their defined service areas, place competitive bids, and manage their assigned harvesting tasks from a dedicated dashboard.
- Location-Based Map View (2D): An interactive 2D map, powered by Mapbox GL JS, visualizes all available tree listings with location markers, allowing contractors to efficiently discover nearby harvest opportunities.
- WhatsApp Notification Integration: Deliver real-time, critical updates and communication for listing activities, bids, and contracts to both tree owners and contractors via integrated WhatsApp alerts.
- Image Upload & Verification: Facilitate secure uploading of multiple tree and harvest photos by users with automated compression, providing visual evidence for quality assurance during transactions.
- Multilingual User Interface: Provide a comprehensive user experience with full support for both Malayalam and English, switchable via an in-app language toggle.
- AI-Powered Dynamic Pricing Tool: An AI tool offering tree owners dynamic pricing suggestions for their produce, leveraging real-time Kerala market data, tree characteristics, and local demand to optimize sales.

## Style Guidelines:

- Primary Color: A vibrant Mango Yellow (#FDB714), echoing the core produce and sunny Kerala landscape. This bold color draws attention to key elements while maintaining a fresh feel. (HSL: 42°, 99%, 60%)
- Background Color: A light and warm Cream/Ivory (#FFF8DC) provides a clean canvas. Its hue subtly aligns with the primary color (45° vs 42°), contributing to a cohesive, light design scheme. (HSL: 45°, 97%, 97.5%)
- Accent Color: An earthy Laterite Red (#C1440E), representing the rich soil of Kerala, used for calls-to-action and important highlights. This color is analogous to the primary and offers strong contrast in brightness and saturation for visual impact. (HSL: 16°, 83%, 40%)
- Headline Font: 'Space Grotesk' (sans-serif), chosen for its modern, slightly technical yet approachable feel, suitable for clear, impactful headings. Body Font: 'Inter' (sans-serif), offering excellent legibility and a neutral, professional appearance for longer text blocks.
- Custom icons featuring natural elements like mangoes, leaves, and avatars with expressive gestures. These elements should reinforce the 'Fresh' brand, contributing to a 'Minimalist but Joyful' aesthetic, as mentioned in the design principles.
- Clean, mobile-first layouts prioritize essential information and provide easy navigation, featuring prominent bottom navigation and accessible touch targets (min. 44x44px). Swipe gestures will be incorporated for common actions like archiving or saving listings, ensuring a thumb-friendly experience.
- Engaging animations including a hero 'Mango Drop' intro sequence (with realistic physics, gentle bounce, leaves fluttering, and splash forming the wordmark). Transitions will be smooth (iOS-style slides, scale-up modals), with custom loading states like spinning mangoes and success states such as confetti bursts.
